# Food Court –  Wireframing & Low-Fidelity Design 

## Project Overview
This project presents the low-fidelity wireframes of **Food Court**, a mobile food delivery application designed using Figma. The wireframes focus on the application's structure, user flow, and layout before moving to the high-fidelity design phase.

The objective of this project was to create a simple and intuitive user experience for ordering food through a mobile application.

---

## Objective
To design low-fidelity wireframes that demonstrate the complete user journey of a food delivery application.

---

## Tool Used
- Figma

---

## Screens Included

### 1. Splash Screen
- Logo placeholder
- Application name
- "Let's Start" button

### 2. Login Screen
- Email field
- Password field
- Login button
- Sign Up option

### 3. Home Screen
- Search bar
- Food categories
- Recommended food items
- Bottom navigation menu

### 4. Food Details Screen
- Food image placeholder
- Food name
- Price
- Rating
- Description
- Add to Cart button
- Back button

### 5. Cart Screen
- Selected item details
- Quantity
- Total amount
- Payment options
- Pay Now button

### 6. Order Confirmation Screen
- Order confirmation message
- Order ID
- Estimated delivery time
- Track Order button
- Back to Home button

---

## User Flow

Splash Screen → Login → Home → Food Details → Cart → Order Confirmation

---

## Design Approach
- Simple and clean layouts
- Clear navigation between screens
- Consistent placement of elements
- Focus on functionality rather than visual aesthetics
- User-centered design approach
